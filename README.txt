Unicode Input by Name
http://code.google.com/p/unicode-input-by-name/

Running (Source)

If you downloaded the source code, you should have wxPython (2.8.x.x) installed
in order to run UIbN. Just execyte uibn.pyw.

Installing and running (Windows)

If you downloaded the Windows binary version, unzip the files to
a writable place in your disk. Run uibn.exe to start the program.
It automatically hides itself. Click the notification are icon or press
Alt+"+" (Numeric pad +) to show the program.
